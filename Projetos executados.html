Máquina Automática para carga e descarga de tijolos

Esta máquina foi projetada para uso específico de uma indústria cerâmica do RS. Sua função é descarregar os tijolos da vagoneta e carregar de maneira agrupada em um pallet, de maneira totalmente automática, com funções complexas e receitas para cada tipo de material com tecnologia SEW.

Painel remoto para operação de máquinas

Este painel foi desenvolvido para operação remota de um automatismo de descarga com pinça de uma indústria cerâmica do RS. Onde o operador possui todos as configurações e visualizações do estado da máquina que opera de maneira automática com tecnologia Weintek.

Painel principal de automatismo de descarga com pinça

Este painel foi desenvolvido para compor todo o sistema de potência, proteções e segurança de um automatismo de descarga com pinça de uma indústria cerâmica do RS. Contempla NR12 categoria 4, acionamentos por inversores SEW para cada motor e CLP com controle via comunicação SBus SEW.

Painel de potência e comando de um soprador de 75CV

Este painel foi desenvolvido para compor todo o sistema de potência, comando, proteções e segurança de um sistema de acionamento para um soprador em uma indústria de couros do RS. Contempla NR12 categoria 4, acionamentos por inversor WEG para para o motor principal, com ajuste de velocidade via IHM na porta do painel, além de acionamentos com disjuntor-motor reservas para futuras ampliações.
